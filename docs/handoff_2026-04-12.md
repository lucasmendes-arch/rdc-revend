# Handoff — Catálogo: Refino Visual P3 (cards, miniaturas, categorias)

_Data: 2026-04-12_
_Autor: Claude (assistido por Lucas)_
_Contexto: sessão de refinamento visual incremental — sem alterações de backend_

---

## 1. Objetivo

Refinar a apresentação visual do catálogo mobile: imagens de produto no grid desktop, cards de pacotes, badges, títulos de seção e filtros por categoria.

---

## 2. Diagnóstico

- Grid desktop: imagens em container landscape (altura fixa 185px) com `object-contain` → portrait products deixavam margens laterais. Mobile era square, visual inconsistente.
- PackageCards: miniaturas pequenas (w-8), card com `flex-1` spacer empurrando botão para baixo, card muito alto, botão "Ver + detalhes" redundante, margem negativa desalinhava com outros blocos.
- Badges de produto (Vegano/Profissional): aprovadas → revertidas → removidas totalmente por decisão do usuário.
- Título "Seleção dos Mais Vendidos": tipografia divergia dos títulos de categoria carousel.
- CategoryBubbles: gap excessivo entre botões, scroll sem `scroll-pl` fazia snap ignorar padding e o primeiro botão saía da borda.

## 4. O que foi implementado

### Catálogo — Grid de produtos
- Container de imagem trocado de altura fixa para `aspect-square` → produto sempre em moldura quadrada igual ao carrossel
- `object-contain` preservado (produtos com fundo branco + `mix-blend-multiply`) com `scale-[1.1]` em repouso e `scale-[1.2]` no hover
- Badges de produto removidas de todos os cards (grid + carousel)

### PackageCards
- Card inteiro é clicável (`onClick → setDetailsPkgId`) — abre modal de detalhes
- CTAs com `e.stopPropagation()` para não conflitar com click do card
- Botão "Ver + detalhes" removido
- Miniaturas mobile: `w-8 h-8 → w-14 h-14`, overlap `-space-x-4 → -space-x-5`, texto `+N` `text-[10px] → text-sm`
- Textos do card: nome `14px → 18px`, descrição `10px → 13px`, contagem `11px → 14px`, retorno estimado `10px → 13px`, preço `18px → 22px`
- Padding unificado `p-4`, sem `flex-1` spacer
- Wrapper simplificado: `mb-4` simples (sem margem negativa), `px-4` alinhado com carrossel

### CategoryBubbles
- Gap entre botões reduzido `gap-4 → gap-2`
- `scroll-pl-4` adicionado para que snap respeite o padding esquerdo

### Título de seção
- "Seleção dos Mais Vendidos": `text-[14px] font-bold → text-[15px] sm:text-lg font-black` (alinhado ao estilo h2 dos carrosséis)
